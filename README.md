**Visit:** https://jose-vargas-git.github.io/roscoe-med-generator/

1. Click "Generate New Schedule"
2. Set start/end dates
3. Add caregiver info (optional)
4. Click "Build & Print"
5. Save as PDF or print

**On iPhone:** Add to Home Screen for easy access.
